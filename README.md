# Ecommerce Flower Shop

A straightforward front‑end website for an online flower store.

## Technologies

- HTML  
- CSS  
- JavaScript (jQuery)  
- Bootstrap 5  
- Owl Carousel 2  
- AOS (Animate On Scroll)

## Features

- Responsive layout  
- Product carousel  
- Scroll‑in animations  
- Contact form  
- Customer reviews
